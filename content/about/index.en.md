I'm **Faiq Najib** — a **Developer, Builder, and Educator** based in Jombang, East Java. I build scalable backend systems, develop digital products, and share knowledge through teaching.

## Developer

Backend engineer with a primary focus on **Go (Golang)** and **PostgreSQL**. Experienced in building high-traffic systems — from a GPS fleet tracking platform with 370+ API endpoints to a SaaS platform for European clients.

- **Languages**: Go, PHP, SQL
- **Databases**: PostgreSQL, TimescaleDB, MySQL, Redis
- **Infrastructure**: Docker, Linux, Nginx, CI/CD, Apache Kafka
- **Specializations**: REST API & microservices, legacy migration (PHP → Go), database optimization

See my work on the [Projects](/en/projects/) page.

## Builder

I don't just write code — I **build products**. PT Akordium Digital Berkah is my sole proprietorship focused on digital solutions for real businesses. Its first product, MIS-APAR, digitized an entire fire extinguisher business operation that was previously manual.

I also have experience delivering freelance projects across various sectors — from construction (inventory management) to banking (process automation & Open API portal).

Need a solution for your business? Check out the [Services](/en/services/) page.
