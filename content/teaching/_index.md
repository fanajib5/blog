Saya percaya bahwa berbagi pengetahuan adalah cara terbaik untuk belajar. Selain bekerja sebagai developer, saya juga aktif mengajar dan mentoring.

## Topik yang Saya Ajarkan

- **Go Programming** — dari dasar hingga concurrency patterns dan microservices
- **Backend Development** — REST API, database design, authentication & authorization
- **Database & SQL** — PostgreSQL optimization, query tuning, schema design
- **DevOps Basics** — Docker, CI/CD, deployment strategies
- **Legacy Migration** — strategi migrasi dari PHP ke Go

## Format Pengajaran

| Format | Deskripsi |
|--------|-----------|
| Workshop | Sesi intensif 1-2 hari dengan hands-on practice |
| Guest Lecture | Sesi kuliah tamu di universitas atau komunitas |
| Mentoring | Pendampingan 1-on-1 atau tim secara berkala |
| Online Session | Kelas daring via Zoom/Google Meet |
